## komen untuk ubuntu
 ``
 ls
 `` 
untuk melihat folder atau file 

``
cd
`` 
untuk keluar dan masuk ke dalam file atau folder 

``
touch 
``
untuk membuat suatu file 

``
nano (nama_file)
``
untuk mengedit isi file 

``
rm -rf (nama file/folder)
``
untuk menghapus file/folder

`` 
mv (nama folder/file)
``
untuk mengganti nama file/folder

## komen untuk docker 
``
docker compose up -d --buid
``
untuk melakukan buil image pada docker 

``
docker compose up -d 
``
untuk menjalankan image docker 

``
docker compose down 
``
untuk menghapus image docker 

```
jika sudah melakukan build atau menjalankan image docker jangan lupa untuk menghapus nya dengan mengetikkan (docker compose down)
```

## komen untuk git hub

``
git add
``
untuk melakukan perubahan pada repo

``
git commit -m "perubahan"
``
untuk commit perubahan (klamat perubahan opisional bisa di ganti)

``
git push 
``
untuk melakukan upload pada git hub
